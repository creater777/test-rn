# Тестовая задача

<p>Написана с использованием expo</p>

## Установка
```sh
git clone https://github.com/creater777/test-rn
cd test-rn
yarn install
```

## Запуск
<p>Приложение использует apikey githab, поэтому в файл .env нужно добавить его значение</p>

```sh
API_KEY=<ваш ключ>
```

<p>Потом запустить</p>

```sh
yarn android
```
